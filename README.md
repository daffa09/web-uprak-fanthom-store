# Fanthom Store

This is my website store, build with HTML, CSS an PHP.

### Prerequisites

- xampp or wampp
- php 7.4.1 or higher

### Setup

- clone this repository
- open xampp or wampp
- open phpmyadmin
- create new database with name `fathom_store`
- import database from `fathom_store.sql`
- open browser and type `localhost/fathom_store`
